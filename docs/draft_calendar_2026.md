# 2026 NBA Draft Calendar & Feature Readiness Plan

**Created:** March 31, 2026

---

## Key Dates

| Date (approx.) | Event | Why It Matters |
|---|---|---|
| **Early April** | Final Four / Championship | Peak casual fan interest in prospects |
| **Mid-April** | Early entry deadline | Player pool finalizes; prospect lists firm up |
| **Late April - May** | Pro Days | Workout data starts flowing |
| **Mid-May** | NBA Draft Combine | Major new data drop (measurements, athleticism) |
| **Mid-May** | Draft Lottery | Pick order set — mock drafts get real |
| **Late June** | NBA Draft (Night 1 & 2) | The main event |

---

## Feature Priorities by Date

### Now → Early April (Final Four window)
- Core player pages, news, and comps are live — site should be solid for the traffic bump.
- **Combine Score UI tickets (CS-01 through CS-06)** — ship these so the stats experience is complete before combine data arrives.

### April → Mid-May (Pre-Combine)
- **Consensus Mock Draft page** — aggregated consensus view ready before the lottery would be a differentiator.
- **Player comparison page (UI)** — the API exists; surfacing a polished head-to-head compare page drives shareability and debate traffic.

### Mid-May (Combine week)
- **Combine data ingestion pipeline** — measurements, agility, shooting data need to flow in quickly. Single biggest data event before the draft.
- **Combine leaderboards** on the stats page should be prominent and shareable.

### Post-Lottery → Draft Night (late May - late June)
- **Share cards polished** — peak social sharing season. Player, compare, and consensus cards should be pixel-perfect.
- **Draft night experience** — even a simple "follow along" view showing picks vs. consensus would be sticky.
